# Dikshant Choudhary — Developer Portfolio

A personal portfolio website built with **React + Vite**, showcasing my projects, skills, and achievements as a Full Stack Developer.

🔗 **Live Site:** [dikshantportfolio-six.vercel.app](https://dikshantportfolio-six.vercel.app)

---

## Sections

| Section | Description |
|---|---|
| **About Me** | Introduction, background, and what I'm currently working on |
| **Skills** | Technologies and tools I work with across frontend, backend, and databases |
| **Projects** | Showcase of full-stack and backend projects with live demo links |
| **Achievements** | Hackathon results, job simulations, and certifications |
| **Contact** | Contact form to reach out directly |

---

## Features

- **Smooth animations & transitions** — section reveals and hover effects for a polished feel
- **Dark mode toggle** — user preference persisted across sessions
- **Fully responsive** — optimized for mobile, tablet, and desktop viewports
- **Live project links** — each project card links directly to the live demo and GitHub repo

## Getting Started

### Prerequisites

- Node.js v18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Dikshant0083/your-portfolio-repo-name

# Navigate into the project
cd your-portfolio-repo-name

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be running at `http://localhost:5173`

### Build for Production

```bash
npm run build
```
